# Melee Island Prelude – v0.3.3

Current state: Playable docks slice with visuals (verbs, hotspots, inventory, narration, hints, glitch, cursor trail, combine swirl).

Run:
pip install -r requirements.txt
python engine/game_manager.py

Controls:
• Left-click verb → select (cursor changes icon)
• Left-click hotspot → interact (pulsing rects on hover)
• H → show hint
• F1 → toggle debug outlines
• Mouse = Interact (with trail effect)

See /working-state/changelog.md for progress
